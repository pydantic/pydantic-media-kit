# pydantic-media-kit

This repository is a media kit for Pydantic. It contains bio, headshot, and official links for Pydantic team members and the company, intended for use by press, event organizers, and partners who need approved assets.

## Contents
- Per-person folders (e.g. `Samuel/`) with `bio.md` containing a short bio, socials, and a headshot.
- Headshot image files at the repo root.
- Official Pydantic social links below.

## Pydantic Logfire
From the team behind the ubiquitous Pydantic Stack, Pydantic Logfire is a full-stack observability platform for AI applications covering logs, metrics, traces, evals, prompt management, and more. By letting coding agents query traces with SQL to understand how an application operates in production, Logfire closes the loop making AI development a supervised learning challenge.
